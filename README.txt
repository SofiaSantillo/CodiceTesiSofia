CausalNex:
- È una libreria Python per l'inferenza causale e l'apprendimento automatico.
- Permette di costruire grafi causali (DAG) che rappresentano le relazioni causali tra variabili.
- Consente di fare inferenza causale (ad esempio, analizzare gli effetti di un cambiamento su una variabile) e modellazione predittiva.
- È utile per capire le relazioni causali ("Perché") tra le variabili, non solo per fare predizioni ("Come").

NOTEARS:
- È un algoritmo di scoperta causale che fa parte di CausalNex.
- Apprende automaticamente la struttura di un DAG da dati osservativi, identificando le relazioni causali tra variabili.
- Ottimizza una funzione obiettivo per trovare un DAG che spiega meglio le dipendenze nei dati.
- È utile per scoprire automaticamente le relazioni causali, anche non lineari, tra variabili, senza conoscenze preesistenti.
